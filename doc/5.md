要完整的描述一个内存单元，需要两种信息：

1. 内存单元的地址
2. 内存单元的长度（类型）

用 [0] 表示一个内存单元时，0 表示单元的偏移地址，段地址默认在 ds 中，单元的长度由具体指令中的其他对象给出。

[bx] 同样表示一个内存单元，偏移地址保存在 bx 寄存器中。

### loop 

loop 指令的格式： 

    loop 标号

CPU 执行到 loop 指令的时候，要进行两步操作：

1. (cx) = (cx) - 1
2. 判断 cx 中的值，不为零则转至标号出执行程序，如果为零则向下执行

```asm
    assume cs:code

    code segment

        mov ax, 2
        mov cx, 11
    s:  add ax, ax
        loop s

        mov ax, 4c00h
        int 21h
    code ends

    end

```
